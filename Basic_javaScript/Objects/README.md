
# Objects

### for creating object

````
var student = {
    name : "shubham",
    age : 24,
}

console.log(student);

````

**Two Ways to access the Objects**
- bracket : objNane["key"];
- dot notation : objName.key;

````
ex:
 student["name"];   // shubham
 student.age;     // 24

````

**for updating the existing key of value**

````
student.name="Ankit";

````

**for adding any new key**

````
student.city="Patna";

````
